# AGENTS

## Project purpose
- This folder contains a Chrome-friendly CRM SPA that stores contacts in `localStorage`.
- The UI is built with plain HTML/CSS/JS (`index.html`, `styles.css`, `app.js`), and the CSS now targets a teal, glassy aesthetic.

## Key instructions for future agents
- Maintain the existing HTML structure so the JavaScript selectors keep working.
- Keep writing in UTF-8 and prefer ASCII characters unless a previous file already requires Unicode.
- Preserve the localStorage-based persistence logic (`app.js`) when changing the interface.
- All styling tweaks should stay inside `styles.css`; the file already defines CSS variables for color references.

## Skills and tooling notes
- No additional Codex skills are required for this simple UI/codebase.
- Manual browser testing of `index.html` in Chrome is recommended after UI changes.
